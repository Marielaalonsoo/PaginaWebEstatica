# Tienda Online

Practica de página web estática desarrollado con HTML y CSS.

## Descripción

Esta web simula una pequeña tienda online de productos tecnológicos.  
Incluye varias páginas interconectadas, diseño con CSS y contenido multimedia.

## Contenido de la web

La página está formada por varias secciones y páginas:

- **Inicio**: presentación de la tienda y producto destacado
- **Productos**: muestra de productos disponibles
- **Carrito**: resumen de compra
- **Contacto**: formulario de contacto para el usuario

## Tecnologías utilizadas

- **HTML5**
- **CSS3**
- **GitHub Pages** para el despliegue

## Características

- Estructura en varias páginas enlazadas entre sí
- Menú de navegación
- Uso de imágenes 
- Formulario web
- Estilo con CSS
- Página publicada en internet con GitHub Pages
